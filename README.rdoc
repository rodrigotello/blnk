== BLNK

BLNK is a blank application in Rails to launch fast prototypes skiping the configuration.

It include:

*

Pipeline:

* Gemfile
* Static Pages: Home, About, Help
* Devise
* Devise edited: Name
* Footer
* Header
* Home Page
* Clean CSS
* Company Model



Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
